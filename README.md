# 💰 SpendWise - Personal Expense Tracker

A lightweight, privacy-focused, spreadsheet-like web application to log daily expenditures, track budgets, manage upcoming bills, and generate financial PDF reports. 

Built with HTML5, Tailwind CSS, and vanilla JavaScript. Hosted entirely on GitHub Pages as an installable Progressive Web App (PWA).

---

## ✨ Features

* **Spreadsheet-Style Logging:** Quick and frictionless entry for expenses (Amount, Category, Date, and Notes).
* **At-a-Glance Dashboard:** Instant breakdown of your spending habits via a dynamic doughnut chart (powered by Chart.js).
* **Budget Progress Bar:** Visual indicator that changes color (Green → Amber → Red) as you approach your monthly budget limit.
* **📋 Bill Reminders:** A dedicated checklist to track upcoming or recurring bills with quick toggle checkmarks.
* **📥 PDF Report Generator:** Filter your data by date range (Past 7 Days, 30 Days, or All Time) and export a clean, printable expense statement.
* **💾 Data Backups:** Export your financial records into a standard `.csv` spreadsheet file at any time.
* **📱 PWA Enabled:** Install directly onto your mobile phone or desktop device and run completely offline.

---

## 🔒 Privacy First

No databases, no login screens, and no tracking. 
All your financial data is saved directly inside your own browser's `localStorage`. Your data never leaves your device.
